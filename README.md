## Scripts
### Start application
```
npm start
```
### Start tests
```
npm RTL tests
```
### Start WebdriverIO tests
```
npm run wdio
```
### Start Storybook
```
npm run storybook
```
### Start Loki screenshot tests
```
npx loki test
```
